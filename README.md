# Personal Website

My own version of a personal website portfolio.

## Installation

Use [pip](https://pip.pypa.io/en/stable/) the following necessary Django library (3.0.8).

```bash
>> pip install django==3.0.8
>> django-admin --version
   3.0.8
>> cd portfolio
>> python manage.py runserver
```
